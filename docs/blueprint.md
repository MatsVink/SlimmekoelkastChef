## Style Guidelines:

- Primary color: Warm terracotta (#E07A5F) for a cozy and inviting feel.
- Background color: Soft beige (#F8F0E3), a desaturated hue of terracotta, creating a warm backdrop.
- Accent color: Olive green (#3D405B) to complement the warm tones and add depth.
- Body and headline font: 'Alegreya', a serif with an elegant and intellectual feel.
- Use simple, line-based icons related to food and cooking.
- Use Tailwind CSS grid and flexbox for a responsive and modern layout.
- Subtle transitions and animations on button hovers and recipe generation.
